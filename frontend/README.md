# ERP Aroma Frontend (PASO 3)

Frontend React 18 + TypeScript para ERP Aroma Marketing.

## Stack

- React 18
- TypeScript
- Tailwind CSS
- React Router
- Axios
- React Hook Form
- Recharts

## Modulos/paginas

1. Login
2. Dashboard
3. CRM Clientes
4. CRM Leads
5. Operaciones
6. Inventario
7. Quejas
8. Reportes

## Ejecutar

```bash
cd frontend
cp .env.example .env
npm install
npm run dev
```

## API

`VITE_API_BASE_URL=http://localhost:3001/api`
